## Responsive Design

### Desktop View

- screen width > 1000px
- Collapsable sidebar on the left
- Sticky content header on top
- Project cards are displayed under 3 columns


### Tablet View
- 550px < screen width < 1000px
- Collapsable sidebar on the left
- Sticky content header on top
- Project cards are displayed under 2 column

### Mobile View
- screen width < 550px
- Sticky content header on top
- Sticky bottom navigation
